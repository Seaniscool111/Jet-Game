# Jet-Game
Designed and extended a 2D game engine in Python featuring physics-based movement, dynamic difficulty scaling, state management, and persistent high-score tracking

## Features
- Physics-based movement (acceleration, friction, velocity)
- Multiple game states (start, pause, gameplay, game over)
- Dynamic level progression and difficulty scaling
- Persistent high-score tracking using file I/O
- Real-time sound effects and background music

## Technologies
- Python
- Pygame

## Controls
- Arrow Keys — Move
- Enter — Start / Restart
- ESC — Pause / Exit

## How to Run
```bash
pip install pygame
python main.py
